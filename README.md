# 🍽️ Savorly

A modern and responsive recipe browsing application built with React. Explore delicious meals, view recipe details, and discover dishes from around the world.

## 🔗 Links

- **Live Demo:** https://savorly-nu.vercel.app

## ✨ Features

- Browse recipes
- View detailed meal information
- Responsive design
- Modern UI with Tailwind CSS
- Data fetched from FreeAPI

## 🛠️ Tech Stack

- React
- Tailwind CSS
- JavaScript
- FreeAPI

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/amnk11/savorly.git
```

Install dependencies

```bash
bun install
```

Start the development server

```bash
bun run dev
```
